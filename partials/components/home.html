Technical Query Analyser is a simple web-app where users can Ask or Answer questions related to Programming languages. User should sign-up to Ask/Answer, after sign-in the page is redirected to dashboard where user can view questions and answers. If any question is not answered and user has knowledge on it then he/she can click ASK/ANSWER button to update the answer..or If user has new question to post then he/she clicks on ASK/ANSWER button to enter category,title and question.
